# chance-sprite

Discord Dice roller for Shadowrun 5e

WIP

Needs a discord_secret.json in ~/.config/chance_sprite
```json
{
  "discord_token": "YOURBOTTOKEN"
}
```

Currently implemented: 
- basic
  - simple (threshold)
  - extended
  - opposed
  - availability
- magic
  - spell
  - create alchemy
  - summon
  - bind
- other
  - starting cash
