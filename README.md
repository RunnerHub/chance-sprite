# chance-sprite

Discord Dice roller for Shadowrun 5e

WIP

Needs a discord_secret.json in ~/.config/chance_sprite
```json
{
  "discord_token": "YOURBOTTOKEN"
}
```

Currently implemented: simple, extended, opposed, threshold, and spell rolls.
